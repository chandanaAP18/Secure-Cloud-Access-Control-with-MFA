
# 🔐 Secure Cloud Access Control with Multi-Factor Authentication (MFA)
📌 Overview

This project implements a secure cloud-based access control system using Multi-Factor Authentication (MFA) to enhance security beyond traditional password-based login. It protects sensitive data by preventing unauthorized access through an additional OTP verification layer.

🎯 Objectives
Design a secure cloud-based login system
Implement MFA (Password + OTP) authentication
Provide Role-Based Access Control (Admin & User)
Securely store and manage user credentials
Monitor and log login activities for security
🚀 Features
User Registration & Login
Email-based OTP Verification
Role-Based Access Control (RBAC)
Secure Password Storage (Encryption)
Login Activity Tracking
Cloud Deployment Support
